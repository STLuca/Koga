Overview

A programming language and operating system are a developers most important tools, after their rubber Hatsune Miku. Koga is an attempt at making better tools.

What does Koga aim to offer?

  • Standard c/Java syntax
  • Control over the abstract machine with a new friendly "assembly" language
  • Internet native system (IPC = IP)
  • Extremely extendable - most standard features are written using the language
  • Protocols as first class citizens of the system
  • Elegant concurrency model
  • Useful (but limited) generics
  • (Soon) Sensible memory management
